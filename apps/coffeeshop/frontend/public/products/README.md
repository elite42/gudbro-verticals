# Product Images Directory

This directory contains product images for the ROOTS menu.

## Naming Convention
- Images should be named using the product slug: `{product-slug}.jpg`
- Example: `lime-juice.jpg`, `immunity-booster-shot.jpg`

## Supported Formats
- JPG/JPEG (recommended)
- PNG
- WebP

## Recommended Image Specifications
- Size: 400x300px minimum
- Aspect ratio: 4:3
- File size: < 500KB
- Quality: 85%

## Upload Process
Images can be uploaded via the Design System Hub at `/design-system/products/upload`
