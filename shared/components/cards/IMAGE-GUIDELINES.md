# Product Card - Image Guidelines

> **Regole per le immagini dei prodotti nelle card GUDBRO**

## Dimensioni Minime per Tipo Card

| Card Type | Aspect Ratio | Dimensione Minima | Uso |
|-----------|--------------|-------------------|-----|
| **Hero**  | 21:9 | 1200 × 514 px | Banner, spotlight |
| **Tall**  | 3:4  | 600 × 800 px  | Grid Pinterest, carousel |
| **Standard** | 4:3 | 800 × 600 px | Liste principali |
| **Compact** | 1:1 | 400 × 400 px | Liste compatte |
| **Mini** | 1:1 | 300 × 300 px | Suggerimenti |
| **Bottle** | 1:2 | 300 × 600 px | Bottiglie, drink alti, prodotti verticali |

## Regole di Composizione

### 1. Zona Sicura per il Prezzo (Bottom Right)
```
┌─────────────────────────────┐
│                             │
│      SAFE ZONE              │
│      (subject here)         │
│                             │
│                    ┌───────┐│
│   NO TEXT/LOGO     │ PRICE ││
│   IN THIS AREA     │ €XX   ││
└────────────────────┴───────┘
```

**Evitare nel 30% inferiore destro:**
- Testo sovrapposto
- Logo del prodotto
- Elementi importanti

### 2. Badge Area (Top Left)
```
┌───────┬────────────────────┐
│ BADGE │                    │
│ AREA  │                    │
├───────┘                    │
│                            │
│                            │
└────────────────────────────┘
```

**I 100px in alto a sinistra** sono riservati ai badge:
- "Editor's Pick" / "Best Seller"
- "Free cancellation"
- Featured star

### 3. Wishlist Area (Top Right)
```
┌────────────────────┬───────┐
│                    │   ♡   │
│                    │       │
│                    └───────┤
│                            │
└────────────────────────────┘
```

**I 60px in alto a destra** sono riservati al cuore wishlist.

## Stile Fotografico

### DO ✅
- **Soggetto centrato** o posizionato con regola dei terzi
- **Illuminazione naturale** o calda
- **Colori vivaci** ma naturali
- **Profondità di campo** che evidenzi il soggetto
- **Orizzonte dritto** per paesaggi
- **Persone in azione** (non in posa fissa)

### DON'T ❌
- Immagini stock generiche con watermark
- Filtri eccessivi
- Testo embedded nell'immagine
- Loghi sovrapposti
- Bordi o cornici decorative
- Collage di più immagini
- Qualità bassa o pixelata

## Esempi per Vertical

### Tours
- Paesaggi con punto focale chiaro
- Guide/turisti in azione (non posati)
- Monumenti/attrazioni senza folla eccessiva
- Golden hour preferita

### Food/Coffeeshop
- Piatti fotografati dall'alto (flat lay) o 45°
- Sfondo neutro o legno naturale
- Vapore/freschezza visibile
- Ingredienti visibili

### Services
- Persone che usano il servizio
- Ambienti puliti e professionali
- Focus sul risultato, non sul processo

### Bottle/Drinks (Card Bottle 1:2)
- Prodotto centrato verticalmente
- Sfondo neutro o gradient leggero
- Illuminazione laterale per riflessi
- Etichetta visibile (se importante)
- Bicchiere/bottiglia interi, non tagliati
- Composizione verticale senza crop laterale

## Compressione e Formato

```
Formato:     WebP (preferito) o JPEG
Qualità:     80-85%
Max size:    200KB per Standard
             300KB per Hero
Colore:      sRGB
```

## Checklist Pre-Upload

- [ ] Dimensioni minime rispettate
- [ ] Zona prezzo libera (bottom right)
- [ ] Zona badge libera (top left)
- [ ] No testo/logo nell'immagine
- [ ] Qualità sufficiente (no pixel)
- [ ] Soggetto ben visibile
- [ ] Colori naturali
- [ ] File size < limite
